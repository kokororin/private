php.fmt
=======

Thank you for upgrading.

New feature added in configuration file:
{
	"passes": ["AutoSemicolon"],
}


//From:
echo $a

echo $a + $b

// To
echo $a;

echo $a + $b;

Note: this feature is still in beta. Use it carefully.
